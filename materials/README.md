# Локальные материалы

Эта папка предназначена для **локальных копий** файлов курсов (PDF, презентации, mindmap и т.д.).

> **В git не коммитятся.** В репозитории хранятся только ссылки на Google Drive в `content/courses/*.json`.

## Рекомендуемая структура

```
materials/
├── modern-art/          # Современное искусство (опционально)
│   └── урок-01/
│       ├── конспект.pdf
│       └── презентация.pdf
└── izi/                 # История зарубежного искусства
    └── урок-01/
```

Можно также держать файлы в папках `СИ_урок_*` в корне проекта — они тоже игнорируются git.

## Связь с сайтом

Сайт читает ссылки из JSON, например:

```json
{
  "label": "Конспект",
  "href": "https://drive.google.com/file/d/FILE_ID/view?usp=drive_link"
}
```

Подробнее — в [docs/CONTENT.md](../docs/CONTENT.md).
